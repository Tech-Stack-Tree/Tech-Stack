1. **숨어있는 숫자의 덧셈(1)**

```java
class Solution {
    public int solution(String my_string) {
        String[] arr = my_string.replaceAll("[^0-9]","").split("");
        
        int answer = 0;
        
        for(int i=0; i< arr.length; i++) {
            answer += Integer.parseInt(arr[i]);
        }
        
        
        
        return answer;
    }
}
```

replaceAll() → 안에 정규식 사용 + split으로  공백

1. **짝수는 싫어**

```java
class Solution {
    public int[] solution(int n) {
        int[] answer;
        int odd = 0;

            if(n%2 == 0) {
                answer = new int[n/2];
            }else {
                answer = new int[n/2 +1];
            }
        
        for(int i= 1; i<=n; i++) {
            if(i%2 != 0) {
                answer[odd] = i;
                odd++;
            } 
        }
        
        return answer;
    }
}
```

**4.세균증식**

```java
class Solution {
    public int solution(int n, int t) {
        int answer = n;
        
        for(int i=1; i<=t; i++) {
            answer *= 2;
        }
        
        return answer;
    }
}
```

초기 answer을 받은 n으로 초기화 그 뒤로 2씩 곱해줌

**5.가위 바위 보**

```java
class Solution {
    public String solution(String rsp) {
        
        //가위 : 2, 바위 : 0, 보 : 5

        String answer = "";
        
//공백으로 split해주기
        String[] s = rsp.split("");
        
        for(int i=0; i<rsp.length(); i++) {
            if(s[i].equals("0")) {
                answer += "5";
            }else if(s[i].equals("2")) {
                answer += "0";
            }else if(s[i].equals("5")) {
                answer +="2";
            }
        }
        
        return answer;
    }
}
```
