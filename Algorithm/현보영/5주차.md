## 001 숫자 비교하기
### 내 코드
```java
class Solution {
    public int solution(int num1, int num2) {
        
        int answer = 0;
        
        if( num1 == num2 ){
            answer = 1;
            return answer;
        }else{
            answer = -1;
            return answer;
        }
    }
}
```

### 잘 짠 코드
```java
class Solution {
    public int solution(int num1, int num2) {
        int answer = (num1 == num2) ? 1 : -1;
        return answer;
    }
}
```
