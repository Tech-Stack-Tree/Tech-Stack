### parseInt 
- `parseInt()`는 어떤 내장함수 객체에도 속하지 않은 전역에서 사용할 수 있는 내장 함수
- 문자열 자료형을 숫자로 변환 가능
- 두개의 인자를 받는다
```js
parseInt(값,진수);
```

```
console.log(parseInt('15',10));
```

### parseFloat
- 어떤 내장 함수 객체에도 속하지 않은 전역에서 사용 가능
```js
console.log(parseInt('5.55'));
```
